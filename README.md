# Test App

My CV in Reat + GraphQL + node.js mode (-:

## Getting Started

* MeteorJS alive! (-: :

          curl https://install.meteor.com/ | sh

* Clone repo:

          git clone https://github.com/n0ne/cv.git

* Change folder :

          cd cv/

* Install packages:

          meteor yarn install  or meteor npm install

* Run app:

          meteor --port 3020

* Open in browser:

          http://localhost:3020
